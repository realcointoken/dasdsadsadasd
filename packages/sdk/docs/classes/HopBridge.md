**`Desc`**

Instantiates Hop Bridge.
Returns a new Hop Bridge instance.

**`Example`**

```js
import { HopBridge, Chain, Token } from '@hop-protocol/sdk'
import { Wallet } from 'ethers'

const signer = new Wallet(privateKey)

```
