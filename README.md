## Quickstart

Install dependencies & link packages

    npm install
    npm run bootstrap

Run frontend app in development

    cd packages/frontend
    REACT_APP_NETWORK=mainnet npm run dev

Visit [http://localhost:3000/](http://localhost:3000/)
